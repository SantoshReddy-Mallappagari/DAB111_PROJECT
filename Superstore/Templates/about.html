About the Dataset

Columns and Data Types:

This dataset contains 9,994 entries (rows) and 13 columns. Below is a detailed description of each column and its data type:

Column Description Data Type
Ship Mode Mode of shipment (e.g., "First Class", "Standard Class"). object (categorical)
Segment Market segment of the customer (e.g., "Consumer", "Corporate"). object (categorical)
Country Customer’s country (e.g., "United States"). object (categorical)
City Customer’s city. object (categorical)
State Customer’s state or region. object (categorical)
Postal Code Customer’s postal or zip code. int64 (numeric)
Region Customer’s region (e.g., "North", "South"). object (categorical)
Category General product category (e.g., "Furniture", "Technology"). object (categorical)
Sub-Category Detailed product subcategory (e.g., "Chairs", "Laptops"). object (categorical)
Sales Total sales value of the order (monetary value). float64 (numeric)
Quantity Number of items purchased. int64 (numeric)
Discount Discount applied to the order (percentage). float64 (numeric)
Profit Profit earned from the order (could be positive or negative). float64 (numeric)

Data Types Summary:

Categorical: Ship Mode, Segment, Country, City, State, Region, Category, Sub-Category (object type).

Numeric: Postal Code, Quantity (int64 type); Sales, Discount, Profit (float64 type).
